About This Project

A suite of automated tools that generate realistic, end-to-end business process data across SAP S/4HANA IDES. Each skill covers a complete business workflow — from master data creation through document posting — and includes Java programs (via JCo RFC), Claude Code skills for AI-assisted operation, and comprehensive HTML documentation. All processes run from a local Mac against the remote SAP system via RFC, with no SSH into SAP servers required.

📦
Order to Cash (OTC)
Sales & Distribution
Complete SD workflow: customer master creation, sales order, delivery, picking, packing, goods issue (PGI), billing document, and accounting entry. Generates realistic sales data across multiple sales orgs and distribution channels.
GitHub Documentation
🛒
Procure to Pay (P2P)
Materials Management
Full MM procurement cycle: vendor master, purchase requisition, purchase order, goods receipt (MIGO), invoice receipt (MIRO), and payment posting. Covers standard and consignment procurement with three-way matching.
GitHub Documentation
⚙
Plan to Produce (PP)
Production Planning
End-to-end PP workflow: BOM creation, routing, planned order conversion, production order release, goods issue to production (261), confirmation, and goods receipt from production (101). Handles co-products and by-products.
GitHub Documentation
📊
Material Requirements Planning (MRP)
Supply Chain Planning
MRP workflow automation: MRP area configuration, planning run execution (MD01/MD02), planned order creation, conversion to production/purchase orders, and stock/requirements list analysis. Bridges planning with P2P and PP execution.
GitHub Documentation
🛠
Housekeeping & Job Scheduler
System Administration
Automated data health maintenance: MM period opening (monthly job via XBP BAPIs), FI posting period monitoring, Material Ledger period extension, number range tracking. Keeps all business processes running by maintaining critical SAP config tables.
GitHub Documentation
🔌
CDS View Extraction
Metadata & SQL Translation
8-phase pipeline that extracts CDS view metadata, dependencies, field definitions, and SQL definitions from SAP via custom RFCs. Resolves SQL-only views via BFS, translates HANA SQL to BigQuery/Snowflake/Databricks ANSI SQL. Powers the Fivetran SAP connector.

Architecture

All tools connect to SAP S/4HANA IDES (sapidess4, system number 03, client 100) via SAP JCo 3 RFC from a local Mac. Direct HANA SQL access is available through an SSH tunnel to port 30015 using hdbcli (Python). Each business process skill is a standalone Java program with a Claude Code skill file for AI-assisted operation. The CDS extraction pipeline runs as a Fivetran custom connector, producing Parquet files in GCS that land in BigQuery.
